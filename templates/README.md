# 🌟 Radiant AI Chat

Radiant AI is a smart, friendly personal AI assistant that chats naturally, helps brainstorm ideas, answers questions, and explains topics clearly. Built with **Python** and **Flask**, this AI app provides a warm, helpful, and approachable chat experience for users.

---

## 🚀 Features

- Friendly and helpful AI chat  
- Remembers conversation during session  
- Explains topics clearly and concisely  
- Easy to run locally or on Replit  
- Clean and mobile-friendly chat interface  

---

## 💻 Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **AI Engine:** OpenAI GPT-4o-mini API
